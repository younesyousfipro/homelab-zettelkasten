# RBAC in DevOps

RBAC is a general concept used across systems to enforce access control.

## Examples
- Linux (users, groups, sudo)
- Cloud IAM (AWS, GCP, Azure)
- Kubernetes RBAC
- Git repository access
- CI/CD permissions

## Purpose
- limit access
- enforce separation of responsibilities
- improve security
