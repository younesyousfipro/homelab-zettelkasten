# Note linking

Notes are connected using explicit links.

Links create relationships between ideas and allow navigation across the knowledge base.

## Types of links

- concept → concept
- concept → procedure
- problem → solution

## Why it matters

- transforms notes into a network
- enables discovery and understanding
- reflects how ideas are connected in practice

## Rule

Links should represent meaningful relationships, not just keywords.
