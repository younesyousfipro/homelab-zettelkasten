# Zettelkasten workflow

A practical way to use Zettelkasten when learning.

## Steps

### 1. Capture

Write a rough note about what you learn.

### 2. Refine

Split into atomic notes.

### 3. Link

Connect to existing notes.

### 4. Reuse

Use notes to produce output (README, docs, projects).

## Key idea

Do not aim for perfect structure.  
Aim for useful connections.
