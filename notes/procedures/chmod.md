# chmod

`chmod` is used to change file permissions.

## Symbolic mode

```bash
chmod u+x script.sh
chmod g-w file.txt
chmod o-r file.txt
``` 

## Octal mode

```bash
chmod 755 script.sh
chmod 644 file.txt
``` 

## Recursive

```bash
chmod -R 755 folder/
```
