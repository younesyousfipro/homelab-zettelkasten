# RBAC in homelab

By default, homelab setups often use full admin access.

## Goal

Simulate real-world environments by restricting access.

## Steps

- create namespaces (dev, prod)
- define roles
- restrict permissions
- test access control

## Why it matters

- understand real-world RBAC usage
- avoid over-permissioning
- practice security principles
