A practical way to use Zettelkasten when learning.

## Steps

1. Capture
- write a rough note about what you learn

2. Refine
- split into atomic notes

3. Link
- connect to existing notes

4. Reuse
- use notes to produce output (docs, README)

## Key idea
Do not aim for perfect structure, aim for useful connections.
