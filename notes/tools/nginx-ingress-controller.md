# NGINX Ingress Controller

NGINX Ingress Controller is a widely used Kubernetes ingress controller based on NGINX.

## Characteristics

- high performance
- mature ecosystem
- widely used in production

## In Kubernetes

- watches Ingress resources
- configures NGINX dynamically
- routes traffic to services

## Why NGINX

- industry standard
- stable and battle-tested
- fine-grained control

## Compared to Traefik

- more mature
- more configuration options
- less automatic than Traefik
