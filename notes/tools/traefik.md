# Traefik

Traefik is a modern reverse proxy and load balancer commonly used as a Kubernetes ingress controller.

## Characteristics

- dynamic configuration
- automatic service discovery
- native Kubernetes integration
- built-in dashboard

## In Kubernetes

- used as an ingress controller
- watches Ingress resources
- routes traffic to services

## In k3s

Traefik is installed by default as the ingress controller.

## Why Traefik

- simple setup
- good defaults
- easy integration in homelab
